# Blender-CompactMAT
Blender plugin for simplified creation and application of materials

A plugin for Blender that simplifies the creation and application of materials without needing to use nodes.

Main tools:

-assign or unassign materials to one or multiple meshes

-create materials with no meshes selected

-one-click setup of displacement

-one-click import multiple textures

-one-click setup of normal or bump map
